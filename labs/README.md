# Labs

Hands-on work — TryHackMe rooms, HackTheBox boxes, homelab setups, CTF writeups.

## Rules I follow
- Never publish writeups for active rooms/boxes that prohibit it.
- Never include real flags or credentials.
- Sanitize anything that could identify clients, employers, or other students.
- All work is on systems I own or have explicit authorization for.

## Layout
- `tryhackme/` — completed rooms
- `hackthebox/` — retired boxes only
- `homelab/` — my own VMs, pfSense, Splunk, etc.
- `ctf-writeups/` — completed CTFs (post-event only)
