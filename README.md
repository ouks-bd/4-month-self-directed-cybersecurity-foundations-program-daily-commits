## Repository structure

```
cybersecurity-journey/
├── 00-roadmap/                 Curriculum, progress tracker, resources
├── 01-threats-and-principles/  Week 1 notes, scenarios, quiz results
├── 02-networking-for-security/ Week 2 + subnetting practice + diagrams
├── 03-identity-and-access/     Week 3
├── 04-cryptography/            Week 4
├── 05-fortinet-fcf/            Week 5
├── 06-cisco-intro-cybersec/    Week 6
├── labs/                       TryHackMe, HackTheBox, homelab, CTF writeups
├── certifications/             Per-cert study plans + completion artifacts
├── tools/                      Cheat sheets for nmap, wireshark, etc.
├── scripts/                    Anything I've automated
└── daily-log/                  Weekly journal entries
```

Each week folder follows the same shape:
- `README.md` — week overview and status
- `notes/` — concept notes in my own words
- `cheat-sheets/` — condensed references
- `scenarios/` — real-world breach analyses
- `diagrams/` — draw.io exports (where applicable)
