# Daily Log

Weekly journal — one file per ISO week (`YYYY-Www.md`).

## Template

```markdown
# Week 23, 2026 (June 1–7)

## Mon
- Studied: [topic]
- Time: 2h
- Wins:
- Stuck on:

## Tue
...

## Friday reflection
- What clicked this week:
- What I'll revisit:
- Next week's focus:
```

Keep entries short. The point is the habit, not the prose.
